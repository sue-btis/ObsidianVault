# üìà Simple Linear Regression (Regresi√≥n Lineal Simple)

La **regresi√≥n lineal simple** es una t√©cnica b√°sica de Machine Learning supervisado usada para predecir un valor continuo (como precio, altura, temperatura) con base en una √∫nica variable independiente.

---
## üß† Idea Principal

La regresi√≥n lineal busca encontrar la mejor **l√≠nea recta** que describa la relaci√≥n entre dos variables:

- **Variable independiente** (X): lo que puedes controlar o conocer (por ejemplo, horas de estudio)
- **Variable dependiente** (Y): lo que deseas predecir (por ejemplo, calificaci√≥n en un examen)

La forma general de la l√≠nea:

$$
y = mx + b
$$

- \( m \): pendiente (slope) ‚Äî indica cu√°nto cambia \( y \) cuando \( x \) aumenta una unidad
- \( b \): intersecci√≥n (intercept) ‚Äî valor de \( y \) cuando \( x = 0 \)

---

## ‚öôÔ∏è Pasos del Modelo

1. **Recolecci√≥n de datos**  
2. **Visualizaci√≥n** con scatter plots  
3. **C√°lculo de los par√°metros $( m$) y $( b$)**  
4. **Predicci√≥n** usando la f√≥rmula $( \hat{y} = mx + b$)  
5. **Evaluaci√≥n del modelo** con m√©tricas como el error cuadr√°tico medio (MSE)

---

## üìê C√≥mo se calcula la mejor l√≠nea

El modelo busca **minimizar el error cuadr√°tico medio** entre las predicciones y los valores reales.

F√≥rmula del error cuadr√°tico medio (MSE):

$$
MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

---

## üíª C√≥digo Python B√°sico

```python
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
import numpy as np

# Datos de ejemplo
x = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)  # Horas de estudio
y = np.array([1.5, 2.0, 3.5, 3.7, 4.5])      # Calificaci√≥n

# Crear modelo
model = LinearRegression()
model.fit(x, y)

# Par√°metros (opcional calcularlos, sirve como elemento visual)
m = model.coef_[0]
b = model.intercept_
print(f"Pendiente: {m}")
print(f"Intersecci√≥n: {b}")

# Predicciones
y_pred = model.predict(x)

# Gr√°fico
plt.scatter(x, y, label="Datos reales")
plt.plot(x, y_pred, color="red", label="L√≠nea de regresi√≥n")
plt.legend()
plt.xlabel("Horas de estudio")
plt.ylabel("Calificaci√≥n")
plt.title("Regresi√≥n Lineal Simple")
plt.show()
